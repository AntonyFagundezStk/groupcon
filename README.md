## Groups Directory (whatsapp, telegram)

### Technologies

- ReactJs
- NodeJs
- ExpressJs
- MongoDB
- Bootstrap (Botswatch theme `Material`)
- JWT

## How to start

- 1. `npm install` inside the GroupCon directory
- 2.  `npm install` inside the Client directory
- 3. to start the project use `npm run dev`
